# Pima Diabetes Prediction using Random Forest

## 📌 Overview

This project focuses on predicting diabetes using the Pima Indians Diabetes dataset.
It demonstrates a complete machine learning workflow using the Random Forest algorithm, from data preprocessing to model optimization and evaluation.

---

## 🎯 Objectives

* Understand and apply Random Forest for classification
* Analyze feature importance (Glucose, BMI, Age, etc.)
* Optimize model performance using hyperparameters
* Evaluate results using metrics like accuracy and confusion matrix

---

## 📊 Dataset

* **Pima Indians Diabetes Dataset** 
* Features include:

  * Glucose
  * BMI
  * Age
  * Insulin
  * Blood Pressure
* Target:

  * 0 → Non-diabetic
  * 1 → Diabetic

---

## ⚙️ Methodology

1. Data preprocessing (handling missing values)
2. Train/test split
3. Model training using Random Forest (CART-based)
4. Hyperparameter tuning (`n_estimators`, `max_depth`, etc.)
5. Model evaluation
6. Feature importance analysis

---

## 🧠 Key Concepts Covered

* Decision Trees (CART)
* Bootstrap & Bagging
* Overfitting and variance reduction
* Feature importance
* Model optimization
